== Description ==

This is a simple Facebook Like Box Plugin. By this plugin you can add a widget for your Facebook page.

== Installation ==

1. Upload "creative-like-box.zip" to the "/wp-content/plugins/" directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. go to "Appearance > Widgets" and you will get a new widget named "Creative Like Box"
4. Just Add the widget in any widget area, just add title, facebook page url, height, width then enjoy.
